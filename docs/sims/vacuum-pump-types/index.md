## About this MicroSim

This animated diagram shows cross-section views of four common vacuum pump types — rotary vane, diaphragm, scroll, and liquid ring — with moving parts to illustrate their operating principles. Summary cards below each pump display key specifications.

## Learning Objective

Students will be able to compare the operating principles of four common vacuum pump types and classify each by its key characteristics.
