## About this MicroSim

This interactive comparison shows the difference between traditional time-based maintenance workflows and modern IoT-enabled condition-based maintenance, highlighting the advantages of continuous monitoring and automated alerting.

## Learning Objective

Students will be able to compare traditional time-based maintenance with IoT-enabled condition-based maintenance and differentiate the key advantages of each approach.
