## About this MicroSim

This MicroSim provides a side-by-side visual comparison of positive displacement (gear pump) and centrifugal pump operation, with classification trees showing common pump types in each category.

## Learning Objective

Students will be able to compare positive displacement and centrifugal pump operating principles and classify common pump types into the correct category.
