## About this MicroSim

This simulated building management system dashboard displays real-time HVAC data for a 4-zone building with 2 air handling units. Switch between scenarios (Normal, Dirty Filter, Refrigerant Leak, Sensor Failure) to practice interpreting system conditions and diagnosing common faults.

## Learning Objective

Students will be able to interpret real-time HVAC monitoring data from a dashboard display and diagnose common system conditions based on sensor readings.
