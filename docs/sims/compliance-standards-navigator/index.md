# Compliance Standards Navigator

This interactive MicroSim helps students classify fluid power system activities under the correct regulatory standards. Explore OSHA, NFPA, ASHRAE, and ISO standards, see which apply to specific maintenance and operation scenarios, and test your knowledge with scenario quizzes.
