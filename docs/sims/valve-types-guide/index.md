## About this MicroSim

This interactive guide shows cross-sections of seven common valve types used in fluid power systems. Hover over any valve to see its function and typical application, or click to see it animate between open and closed positions.

## Learning Objective

Students will be able to identify common valve types by appearance and recognize their function in a fluid power system.
